# Slot Machine in Python

A number slot machine developed in Python as a learning project.

## Description

The program generates three random numbers between 0 and 5. If the three numbers are the same, the player wins. Otherwise, they lose.

## Features

- Random number generation.

- Win/loss check.

- Simple terminal interface.

- Developed entirely with Python.

## Technologies

- Python 3
- `random` module

## How to run

1. Clone this repository:
```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
```

2. Navigate to the folder:
```bash
cd YOUR_REPOSITORY
```

3. Run the program:
```bash
python main.py
```

## Objective

This project was created to practice basic Python concepts, such as:

- Variables
- Data input (`input`)
- Loops (`while`)
- Conditionals (`if` / `else`)
- Random numbers (`random`)
